A # Finite Element Method

A collection of Finite Element Method implementations developed from first principles using MATLAB and Python.

## Repository Goals

This repository documents my journey of implementing finite element methods from scratch, beginning with the mathematical foundations and progressing to advanced nonlinear simulations.

Topics include:

- Mathematical foundations of FEM
- 1D Bar Element
- Steady-State Heat Conduction
- 2D Truss Analysis
- Beam and Frame Elements
- Dynamic Finite Element Analysis
- Nonlinear Finite Element Method
- Phase-Field Modeling (Cahn–Hilliard Equation)

## Programming Languages

- MATLAB
- Python
- FEniCSx
